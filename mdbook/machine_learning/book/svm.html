SVM

SVM was the best model before Neural Network.

Intuition

Let's quickly recap how to represent a hyperplane in n-D space.

In high school, we learned a line in 2D space can be represented as: y=ax+b This is equivalent to: w1​x+w2​y+b=0 For a plane in 3D: w1​x+w2​y+w3​z+b=0 For a hyperplane in n-D: w1​x1​+w2​x2​+w3​x3​+...+wn​xn​+b=0 Which is equivalent to: wTx+b=0 This is the general form of a line, plane, and hyperplane. Here, w∈Rn×1. If n=2, it's a line; if n=3, it's a plane; if n>3, it's a hyperplane.

Alternatively: θTx=0,θ∈Rn+1×1

So, let's first try to investigate SVM in 2-D space.

A line divides points into 2 classes. Thus, our decision function is:
hw,b​=g(wTx+b)

g(z)=1, if z≥0; g(z)=−1, if z<0

Margin

To know the confidence level of our classification result, the first intuition might be: directly input the point's coordinates into the line and use the magnitude and sign of the result to tell the correctness and confidence of our classification.

Suppose the point is (x1​,x2​), and the classification result is y1​.

Here is the definition of funciton margin γ^​=y(wTx+b)

However, this margin is not ideal. If we scale both w and b up, the hyperplane remains unchanged, but the function margin scales up.

So, here comes the definition of geometric margin, which is the distance from the data point to the hyperplane. γ=∥w∥wT(x+b)​

Note: when ∥w∥=1, function margin = geometric margin.

We only care about the min margin when the hyperplane is fixed.

Outliers?

Adding a penalize term for ourtliers γ,w,bmin​bs.t.​21​∥w∥2+Ci=1∑n​ξi​y(i)(wTx(i)+b)≥1−ξi​, i=1,...,mξi​≥1​

  • C: is the egularization parameter we can choose, how much we want to penalize the outliers. Bigger C means we places more emphasis on minimizing the classification error, potentially at the expense of a smaller margin.
  • ξi​: is the slack variable. if ξi​=0, the data point is correctly classifed outside the margin. if ξi​>0, the data point is either misclassified or within the margin.
